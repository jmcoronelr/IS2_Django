Atributos: - id: Identificador único del usuario. - username: Nombre de usuario único. - email: Correo electrónico único. - nombre: Nombre del usuario. - apellido: Apellido del usuario. - is_active: Estado de actividad del usuario. - is_staff: Indica si el usuario tiene permisos administrativos. Métodos: - __str__: Retorna una representación en cadena del usuario (su correo electrónico).
